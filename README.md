<h1>Apple Juice – Landing Page</h1>

Apple Juice is a simple landing page created for promoting a juice manufacturer and for practicing core front-end development skills.
The project focuses on clean layout, basic interactivity, and improving JavaScript fundamentals.

<h2>Technologies Used</h2>

- HTML5
- CSS3
- JavaScript (Vanilla)

Project Purpose
This project was built to:

- Create a basic landing page for juice product promotion
- Practice JavaScript fundamentals and DOM manipulation
- Work with JSON data and dynamic content rendering
- Implement basic UI features such as theme switching

Features

- Light and Dark mode toggle
- Products stored in JSON files
- Dynamic product listing generated from JSON data
- Individual product pages for each product
- Clean and simple UI

How to Run the Project

- This project must be run using a local server due to loading JSON files with JavaScript.

Recommended method: Live Server

- Open the project folder in Visual Studio Code
- Install the Live Server extension
- Right-click on index.html
- Select “Open with Live Server”
- The application will open in your browser and work correctly.

Data Handling

- Product data is stored inside JSON files.
- These files are loaded using JavaScript and dynamically rendered on the landing page.
- Each product has its own dedicated page based on the JSON data.
